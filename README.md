# Streamlit test

This is a simple test of Streamlit.

First create the file `.streamlit/secrets.toml` with the following content:

```toml
host = "my_host"
user = "my_user"
password = "my_password"
database = "my_database"
port = "my_port"
```

Then run the following command:

```bash
streamlit run Inicio.py
```
